# 🎴 יניב! — משחק הקלפים

משחק **יניב** מלא לדפדפן, מותאם לטלפון, בעברית — בלי שרת ובלי הרשמה.

## מצבי משחק

| מצב | איך |
|---|---|
| 📶 **אונליין עם חברים** | כל אחד מהטלפון שלו! יוצרים חדר, משתפים קוד בן 4 תווים (או קישור), וחברים מצטרפים מכל מכשיר. החיבור ישיר בין המכשירים (WebRTC/PeerJS) — אין שרת משחק. |
| 🤖 **נגד בוטים** | 1–3 יריבים ממוחשבים בשלוש רמות קושי. |
| 📱 **באותו מכשיר** | 2–4 שחקנים מעבירים את הטלפון, עם מסך הסתרה בין תורות. |

## איך משחקים (חוקי יניב)

- כל שחקן מקבל 5 קלפים. המטרה: כמה שפחות נקודות ביד.
- בתור שלך: **מסמנים** קלף / זוג / שלישייה / רצף — ואז **לוחצים על הקופה** (לזרוק ולמשוך)
  או **על קלף קצה בערימה** (לזרוק ולקחת אותו). פעולה אחת וזהו.
- כפתור **יניב!** מלווה אותך כל המשחק — אפור כשאי אפשר, ונדלק בזהב כשסכום היד ≤ 7.
- הכרזת יניב? אם למישהו יש פחות או שווה — **אסף!** והמכריז חוטף +30.
- חוק ה-50: נחיתה בדיוק על כפולה של 50 חותכת את הניקוד בחצי 🎉
- הצמדה: משכת מהקופה קלף זהה למה שזרקת — יש 3 שניות להצמיד!

## הרצה מקומית

זה אתר סטטי — אין build:

```bash
python3 -m http.server 8080
# ואז http://localhost:8080
```

## פריסה

כל דחיפה ל-`main` מפרסמת אוטומטית ל-GitHub Pages (ראו `.github/workflows/pages.yml`).
המשחק האונליין דורש HTTPS — GitHub Pages מספק את זה מהקופסה.

### בדיקת אונליין בפיתוח

מצב האונליין משתמש בברוקר הציבורי של PeerJS. לבדיקות מקומיות אפשר להריץ ברוקר משלכם:

```bash
npx peer --port 9000 --path /ps
# ואז לפתוח את המשחק עם ?ps=localhost:9000
```

## טכנולוגיה

- HTML/CSS/JS ונילה — בלי פריימוורקים, בלי תלויות build
- [PeerJS](https://peerjs.com) (מוטמע ב-`vendor/`) — WebRTC למשחק בין מכשירים; המארח הוא "השרת"
- WebAudio לצלילים מסונתזים, PWA manifest לשמירה למסך הבית
